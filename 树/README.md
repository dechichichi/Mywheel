二叉树
每个节点有左右子树和数值
二叉查找树 左子树的节点值小于根节点 右子树的节点值大于根节点 对于每个节点都是如此
这样使得查找的期望时间变成了 O(Log N)

AVL树
带有平衡条件的二叉查找树
平衡树:对于每个节点 左右子树的深度差不超过1
为了保证插入后AVL树能保持平衡
我们引入单旋转与双旋转
由于插入的位置不同 分为左单 左双 右单 右双
单旋转以左单为例
根节点K1 K2=K1->left  
此时K1的左子树深度高于右子树 不平衡 称为左重于右
为了平衡 我们加重右
K2->right=k1->right
把k2的right作为新的k1
k2->left=k1
这样旋转一次 左树的深度-- 右树深度++ 达成了平衡

双旋转是对于节点在内部的情况的 
仅仅一次单旋只能把节点放到外面 无法做到减重
故进行二次单旋 达成平衡
